title: Wormhole MUSD Bridge
topic: developers
---

El MUSD Bridge está impulsado por el protocolo Native Token Transfer (NTT) de Wormhole, que permite transferencias cross-chain seguras manteniendo la fungibilidad del token. Para más información sobre el MUSD Bridge, consulta la página [MUSD Bridge](/es/docs/users/musd/musd-bridge).
