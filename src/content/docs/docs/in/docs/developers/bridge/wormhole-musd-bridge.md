MUSD ब्रिज Wormhole के नेटिव टोकन ट्रांसफर (NTT) प्रोटोकॉल द्वारा संचालित है, जो टोकन की विनिमेयता बनाए रखते हुए सुरक्षित क्रॉस-चेन ट्रांसफर को सक्षम करता है। MUSD ब्रिज के बारे में अधिक जानकारी के लिए, [MUSD ब्रिज](/docs/in/docs/users/musd/musd-bridge) पृष्ठ देखें।
