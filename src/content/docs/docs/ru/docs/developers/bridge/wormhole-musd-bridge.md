title: Wormhole MUSD Bridge
topic: developers
---

Мост MUSD работает на основе протокола Wormhole Native Token Transfer (NTT), который обеспечивает безопасные кросс-чейн переводы при сохранении взаимозаменяемости токенов. Для получения дополнительной информации о мосте MUSD смотрите страницу [Мост MUSD](/docs/ru/docs/users/musd/musd-bridge).
