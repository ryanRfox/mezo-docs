---
title: Wormhole MUSD Bridge
topic: developers
---

Le pont MUSD est alimenté par le protocole Native Token Transfer (NTT) de Wormhole, qui permet des transferts inter-chaînes sécurisés tout en maintenant la fongibilité des tokens. Pour plus d'informations sur le pont MUSD, consultez la page [Pont MUSD](/docs/fr/docs/users/musd/musd-bridge).
