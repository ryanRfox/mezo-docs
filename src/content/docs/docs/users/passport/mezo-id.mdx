A Mezo ID is a human-readable identifier linked to a wallet on the Mezo network. Instead of sharing a full `0x…` wallet address, users can share their Mezo ID to receive assets.

## How it works

When someone enters your Mezo ID in the [Send form](/docs/users/passport/send-assets), Mezo resolves it to your wallet address automatically. The resolved address is displayed as a confirmation step so the sender can verify the destination before completing the transaction.

The same resolution flow applies when a Bitcoin address is entered — Mezo maps it to the associated wallet on the network.

## Using your Mezo ID

- **Receiving assets** — Share your Mezo ID with other Mezo users instead of your full wallet address.
- **Sending assets** — Enter a recipient's Mezo ID in the Passport Send form. Verify the resolved wallet address before confirming.
