## Antwyr

[Antwyr](https://antwyr.com/) is a collection of browser-based blockchain tools for Mezo.

URL: [https://antwyr.com/](https://antwyr.com/)

- Export blockchain data, including blocks and transactions, from the Mezo network with CSV and TXT download support for analysis or record-keeping.
- Decode raw transaction calldata so you can inspect function names, parameters, and token amounts before signing.
- Generate a live heatmap that visualizes currency volumes transacted per block on the Mezo network in real time.

## Matchbox

[Matchbox](https://matchbox.markets) is a matching market that connects veBTC holders with veMEZO holders on Mezo, creating a marketplace for voting, incentives, and boost.

URL: [https://matchbox.markets/](https://matchbox.markets/)

- veBTC holders can offer incentives to attract veMEZO votes and boost veBTC positions (up to 5x).
- veMEZO holders can use voting power on gauges with attractive incentives to earn rewards.
- Read the [Matchbox docs](https://matchbox.markets/docs) for guides, concepts, and FAQs.

## MezoAgent

[MezoAgent](https://github.com/LowPolyDuck/MezoAgent) is a LangChain-powered Web3 AI agent that facilitates plain-English BTC and mUSD transactions on Mezo.

URL: [https://github.com/LowPolyDuck/MezoAgent](https://github.com/LowPolyDuck/MezoAgent)

- Send BTC and mUSD to any address using natural language commands instead of manual transaction construction.
- Swap mUSD for BTC via DumpySwap directly through conversational prompts.
- Built with LangChain and web3.py, using structured output parsing to extract transaction parameters from freeform text.

## Mezo Tools

[Mezo Tools](https://mezotools.cc/) is an analytics and monitoring dashboard for MUSD and collateral positions.

URL: [https://mezotools.cc/](https://mezotools.cc/)

- Track system-level collateralization and monitor MUSD health metrics.
- Review individual troves, redemptions, and liquidation activity.
- Use it for higher-frequency monitoring beyond the core app views.

## veBoost Calculator

[veBoost Calculator](https://boost.mallard.sh/) is a calculator tool for estimating veBoost outcomes on Mezo.

URL: [https://boost.mallard.sh/](https://boost.mallard.sh/)

- Estimate boost multipliers for veBTC and veMEZO positions before committing to a lock.
- Model different lock amounts and durations to optimize your boost strategy.
- Use it as a companion tool alongside the Mezo Earn lock and vote workflows.

## Vezo

[Vezo](https://vezo.exchange/) is a marketplace for veNFT positions, including both veMEZO and veBTC positions.

URL: [https://vezo.exchange/](https://vezo.exchange/)

- Browse veMEZO and veBTC position listings and marketplace activity.
- Buy or sell veMEZO and veBTC positions through a dedicated marketplace interface.
- Use it when you want a secondary market workflow for veNFT positions.
