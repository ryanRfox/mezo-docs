title: Wormhole MUSD Bridge
topic: developers
---

Die MUSD-Brücke wird vom Wormhole Native Token Transfer (NTT)-Protokoll betrieben, das sichere Cross-Chain-Transfers bei gleichzeitiger Wahrung der Token-Fungibilität ermöglicht. Weitere Informationen zur MUSD-Brücke finden Sie auf der Seite [MUSD-Brücke](/docs/de/docs/users/musd/musd-bridge).
