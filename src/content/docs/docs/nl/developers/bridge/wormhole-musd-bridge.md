De MUSD-brug wordt aangedreven door het Wormhole Native Token Transfer (NTT)-protocol, dat veilige cross-chain transfers mogelijk maakt met behoud van token-fungibiliteit. Voor meer informatie over de MUSD-brug, zie de pagina [MUSD-brug](/docs/nl/users/musd/musd-bridge).
