title: Wormhole MUSD Bridge
topic: developers
---

Міст MUSD працює на основі протоколу Wormhole Native Token Transfer (NTT), який забезпечує безпечні крос-чейн переводи зі збереженням взаємозамінності токенів. Для отримання додаткової інформації про міст MUSD, дивіться сторінку [Міст MUSD](/ua/docs/users/musd/musd-bridge).
