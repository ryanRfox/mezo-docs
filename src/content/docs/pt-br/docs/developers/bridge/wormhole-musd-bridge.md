A ponte MUSD é alimentada pelo protocolo Native Token Transfer (NTT) da Wormhole, que permite transferências cross-chain seguras mantendo a fungibilidade dos tokens. Para mais informações sobre a ponte MUSD, veja a página [Ponte MUSD](/pt-br/docs/users/musd/musd-bridge).
